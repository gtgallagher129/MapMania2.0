Getting Started

Welcome to the Guess the National Park Game!

Your goal is to click on the map to guess the location of a national park.

When you think you've found the park, click within the specified distance to win.

The game keeps track of your score, and you win when you've found all the parks.

Click anywhere on the map to start playing!

If you want to skip the game and win instantly, you can use the "Win Now" cheat code. Just click the "Win Now" button located in the game interface. A confirmation dialog will appear to make sure you want to skip the rest of the game and win now. Use this feature wisely!
